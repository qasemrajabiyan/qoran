# راهنمای پنهان‌سازی کد برکت هاب

## یک بار — نصب ابزار

ترمینال را باز کنید (در پوشه پروژه):
```
cd Desktop\qoran
npm install javascript-obfuscator
```

## هر بار قبل از آپلود

```
node obfuscate.js
```

پوشه `dist/` ساخته می‌شود — **این پوشه را آپلود کنید، نه پوشه اصلی**.

## نتیجه

- پوشه `assets/js/` — کد اصلی خوانا (نگه دارید، دست نزنید)
- پوشه `dist/` — کد پنهان‌شده برای آپلود

## چه چیزی پنهان می‌شود؟

- نام API ها (Claude، ElevenLabs، OpenAI)
- نام متغیرها و توابع
- رشته‌های متنی (URL ها، پیام‌ها)
- console.log ها غیرفعال می‌شوند
- debug کردن سخت‌تر می‌شود

## توجه

فایل‌های اصلی در `assets/js/` دست نخورده می‌مانند.
هر بار که کد را ویرایش کردید، دوباره `node obfuscate.js` را اجرا کنید.
